### 即時友善氣象站通知系統(地區性)

1.透過訊息推送方式傳送氣象資訊給用戶.<br/>
2.用戶可自行調整通報時間(小時為間隔)、地點(最多複選三).<br/>
3.利用apk安裝服務軟體(WebView).<br/>

### 可能使用技術(工具)

1.HTML、CSS、JavaScript(jQuery).<br/>
2.MySQL、PHP.<br/>
3.Ajax.<br/>
4.PhotoShop.<br/>
5.Java.<br/>
6.Node.js、npm.


### 硬體

1.PM2.5感測器<br/>
2.溫度、濕度感測器<br/>
3.Arduino、樹莓派三代
